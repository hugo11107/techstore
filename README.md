# TechStore Project Overview

TechStore is an innovative e-commerce platform that offers a wide range of products, from electronics to fashion. Our goal is to provide users with the best online shopping experience.

## Features
- User-friendly interface
- Secure payment options
- Order tracking
- Product reviews and ratings
- Wishlist functionality

## Setup Instructions

To set up the TechStore project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hugo11107/techstore.git
   cd techstore
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.
